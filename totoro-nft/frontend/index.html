TESTING RUNBOOK

建议的端到端测试顺序

  1. 铸造连接拥有 Sepolia ETH 的钱包，以默认 URI 铸造，等待交易确认并记录 Token ID。
  2. 固定价交易使用 Token ID 上架；切换至第二个钱包购买，确认 NFT 所有者已更新。
  3. 拍卖铸造另一枚 NFT 后创建 2 小时拍卖；两个不同买家依次出价，较低出价者提取退款。
  4. 结算时间到期后由任意账户结算，确认最高出价者收到 NFT，卖家和手续费地址收到 ETH。
